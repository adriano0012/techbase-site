Curiosidades sobre HTML e Sites

  • HTML significa HyperText Markup Language.
  • HTML não é linguagem de programação.
  • Todo site começa com a tag html.
  • Sem HTML, não existe site.
  • Navegadores apenas interpretam o HTML.
  • HTML funciona até sem internet.
  • Um erro pequeno pode quebrar um site inteiro.
  • Tags abertas precisam ser fechadas.
Mais curiosidades

Muitos problemas em sites acontecem por falta da tag meta charset, causando letras quebradas ou símbolos estranhos.

Outro erro comum é esquecer de fechar tags como p, div ou table.

Como o HTML funciona?

O HTML funciona por meio de tags. As tags dizem ao navegador o que cada parte do conteúdo representa.

Exemplo básico:

<h1>Título do site</h1>
<p>Texto do site</p>

Mesmo isso já é considerado um site.

Ajuda em HTML – Tags Básicas

  • h1 até h6 – Títulos
  • p – Parágrafo
  • a – Link
  • ul / li – Lista
  • table – Tabela
  • form – Formulário
  • input – Entrada de dados
  • textarea – Campo de texto
  • details – Conteúdo expansível

Ajuda em HTML – Problemas Comuns

Texto com símbolos estranhos

Normalmente acontece quando falta a tag meta charset="UTF-8".

Site quebrado sem erro no editor

Pode ser tag não fechada corretamente ou erro de estrutura.

Site funciona no editor mas não no navegador

O navegador é quem realmente interpreta o HTML.

HTML não aparece como esperado

Ordem das tags e aninhamento incorreto podem causar isso.
